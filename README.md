# PSD2UGUI_X
Convert psd file to ugui prefab,  text, image, raw image, button, slider, scroll view, dropdown, toggle, textmeshpro...

psd一键拼UI插件。与其它插件不同，此插件进行了大量优化，降低美术、技术使用门槛。插件在unity中直接把psd文件解析成可编辑的节点树prefab，ui程序可以任意调整或组织层级，点击生成即可得到拼好的uGUI预制体。

若感兴趣可以购买源码Pro版本：[获取超强大的Pro版本插件](https://item.taobao.com/item.htm?id=755447921869)

功能视频教程：[https://www.bilibili.com/video/BV1SVUkBWE2v/](https://www.bilibili.com/video/BV1SVUkBWE2v/)

Pro版功能：
1. 自研的psd解析库，无水印、更轻、更快、功能更稳定。
2. 支持所有uGUI类型、TextMeshPro_GUI以及额外支持Fill Color(纯色)。
3. 支持psd图层引用复用、图片复用、本地图片复用、prefab复用， 支持全局资源复用(跨界面复用)。
4. 智能九宫识别。自动对Sprite精准识别九宫分割，支持纹理识别。
5. 文本描边和阴影效果同步。
6. 强大的编辑器功能，支持增量更新等，对美术、UI程序新手友好。
7. 美术效率工具，批处理PS脚本。
